# BandManager

#### _Angular2 JS CLI App, January 30th, 2017_

#### By _**Adam Zinder**_

## Description

_Manager your own band using Angular2 JS CLI app._

## Setup/Installation Requirements

* _$ git clone https://github.com/azinder1/band-manager_
* _$ npm install_
* _$ bower install_
* _$ ng serve _
* _Open localhost:4200_
